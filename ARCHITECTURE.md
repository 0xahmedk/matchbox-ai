## Key Concepts

### 1. Separation of Concerns

- **Engine**: Pure TypeScript, no React dependencies
- **Integration**: React hooks for state management
- **UI**: Your components (to be built)

### 2. Functional Design

- Immutable operations
- Pure functions for transformations
- Side effects isolated to training

### 3. Performance Optimizations

- Symmetry reduction (98.5% state space reduction)
- Efficient canonical state lookup (O(72) operations)
- Weighted random selection (O(9) operations)

### 4. Extensibility

- Configurable learning parameters
- Pluggable training strategies
- Serializable memory (import/export)
- Framework-agnostic core
